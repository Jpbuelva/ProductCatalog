# Prompts utilizados con IA

**Herramienta utilizada:** GitHub Copilot

Durante el desarrollo de este proyecto se utilizaron los siguientes prompts para asistencia y generación de código:

- "Sugiere estructura de carpetas y archivos para un backend limpio en .NET 8."
- "Implementa un CRUD REST para productos."
- "Agrega validaciones con FluentValidation para ProductDto."
- "Genera un archivo run.md con instrucciones claras para ejecutar el proyecto."
- "Revisa el controlador y sugiere mejoras."
- "Crea pruebas unitarias para ProductService y ProductsController."
- "¿El código que tengo actualmente corresponde a lo pedido?"
- "¿Qué falta por prueba unitaria?"
- "Dame todo organizado para la entrega."
- "Crea el README.md y plan.md para la documentación."
